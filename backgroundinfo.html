Behind the project

    Project description:
    A study on how temperature affected students emotions using an arduino temperature sensor as well as creating a website to display the results.
    environment needed to run the program:
  • Chromebook,
  • a google account,
  • Google Sheets
  • Colaboratory (need the url)
  • github.com,
  • a github account
  • Arduino UNO and attachments (DHT22 sensor, breadboard with attached LED and button)
    • Installed software:
  • the Arduino Create Chrome App (from the Google App Store)

  • In order to make the graphs we used python to transform a google sheets documents data into something visual.
  • All of the data was collected from 2/25/19-3/26/19
