Behind the project

This projects was done using a temperature sensor with arduino software. In order to make the graphs we used python to transform a google sheets documents data into something visual. All of the data was collected from 2/25/19-3/26/19, and this website was made with github. While doing this study there were some setback. While coding the arduino the code wouldn't work a lot and we had a lot of software difficulties.
